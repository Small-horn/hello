4. API 接口说明

后端提供的相关API接口：

  • 发布公告: PUT /api/announcements/{id}/publish
  • 取消发布: PUT /api/announcements/{id}/unpublish
  • 状态说明:
    • DRAFT (草稿) → 显示"发布"按钮
    • PUBLISHED (已发布) → 显示"取消发布"按钮
    • CANCELLED (已取消) → 显示"发布"按钮
    • EXPIRED (已过期) → 显示"取消发布"按钮
